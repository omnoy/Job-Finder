Server requirements setup:
- We use MongoDB Atlas (a hosted MongoDB service option in the cloud) which requires no installation overhead.
- Add an OpenAI API key in the application.properties field.

Client requirements setup:
1. Flutter local installation: https://docs.flutter.dev/get-started/install
2. Setup flutter in IDE: https://docs.flutter.dev/tools/vs-code#installation-and-setup
3. Open main project folder.
4**. Go to lib/main.dart, then click Run > Start Without Debugging in the main IDE window, or press Ctrl+F5
** If you encounter compilation issues: https://dart.dev/tools/dart-fix
